# Sistema AI - Generador de Anexos DTE

Aplicación web en Flask para procesar archivos JSON de DTE y generar anexos en Excel (.xlsm).

## 🚀 Funcionalidades

- Carga múltiple de JSON
- Clasificación automática
- Generación de anexos:
  - Contribuyentes
  - Consumidor final
  - Compras

## 🛠️ Tecnologías

- Python
- Flask
- OpenPyXL
- Docker

## ▶️ Ejecución local

```bash
pip install -r requirements.txt
python app.py
